# Lottery Number Generator

## Overview

This project is a simple web application that generates random lottery numbers. The user can click a button to get a new set of 6 unique numbers between 1 and 45.

## Design and Features

*   **UI:**
    *   A main title "Lotto Number Generator".
    *   A container to display the generated lottery numbers.
    *   A button to trigger the number generation.
*   **Styling:**
    *   Modern and clean design.
    *   Responsive layout for different screen sizes.
    *   Visually appealing number display.
    *   Styled button with hover effects.
*   **Functionality:**
    *   Generates 6 unique random numbers from 1 to 45.
    *   Displays the numbers in the UI.
